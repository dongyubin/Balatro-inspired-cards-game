Privacy Policy

A plain-language summary of data handling on the Balatrio website.

Effective June 12, 2026

Information collected by this website

The Balatrio landing page is a static website. It does not provide user accounts, payment forms, comments, or a database-backed leaderboard. The site code does not intentionally collect names, email addresses, passwords, or payment details.

Hosting and technical logs

The site may be hosted by Vercel or another infrastructure provider. Like most web hosts, that provider can process basic request information such as IP address, device type, requested URL, timestamps, and security diagnostics to deliver and protect the service. Their handling of those logs is governed by their own privacy terms.

Embedded game prototype

The playable prototype is loaded in an iframe from a third-party Yourware domain. Loading that frame sends a normal web request to the external host. The Balatrio landing page does not control the host's server logs or cookies. You may block third-party content in your browser, although doing so can prevent the game from loading.

Cookies and analytics

This version of the Balatrio site does not intentionally set first-party advertising or analytics cookies. If analytics are added later, this policy and the changelog will be updated before the change is treated as part of the public site.

Your choices

Browser controls can clear stored site data, block third-party frames, restrict cookies, or disable JavaScript. The written game guide remains available without interacting with the embedded prototype.
